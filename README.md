INSTALLATION#
1. Make a shelf Tool.
2. Save the shelf tool.
3. Goto the Script Tab. Insert this code below :

import importlib as il
import alembicimporter as ai
appWin = ai.MainWindow()

il.reload(ai)

4. Save the Shelf Tool Again.
